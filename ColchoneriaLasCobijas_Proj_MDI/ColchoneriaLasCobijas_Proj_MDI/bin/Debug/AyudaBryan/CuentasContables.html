Cuentas Contables


Agregar Cuenta Contable a Raíz

Para agregar una nueva cuenta contable a la raíz se debe de escribir el nombre y seleccionar el tipo de cuenta que es, no seleccionar ninguna cuenta del cuadro inferior y presionar el botón de AGREGAR. Si no se selecciona o ingresa algún dato saldrán mensajes de advertencia. La siguiente imagen es el ANTES.

Agregar Cuenta Contable a Nivel Seleccionado

Para agregar una nueva cuenta contable a un nivel seleccionado se debe de escribir el nombre y seleccionar el tipo de cuenta que es, luego seleccionar una cuenta a la cual se va agregar del cuadro inferior y presionar el botón de AGREGAR. Si no se selecciona o ingresa algún dato saldrán mensajes de advertencia. La siguiente imagen es el ANTES.

Si los datos están bien se mostrará el siguiente mensaje.
